# Equity Analyzer

A Python application for analyzing publicly traded companies.

## Project Goals

This project is built to improve my Python and financial analysis skills.

The application will eventually be able to:

- Download stock prices
- Display interactive charts
- Calculate volatility
- Calculate beta
- Show valuation metrics (P/E, EPS, ROE)
- Compare multiple companies

## Technologies

- Python
- pandas
- yfinance
- matplotlib
